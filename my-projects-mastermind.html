Mastermind is a game where you must guess a secret code. One player creates a code using colored pegs, and the other tries to crack it.
You submit a row of colors as your guess.
The code maker provides hints: a black peg for a correct color in the correct position and a white peg for a correct color in the wrong position. juiste kleur op de verkeerde plek.
The goal is to solve the puzzle in as few turns as possible!
